## üõ†Ô∏è Usage

1. **Encrypt Text**:
   - Enter the text you want to encrypt in the input field.
   - Click on the "Encrypt" button.
   - The encrypted text will be displayed.

2. **Decrypt Text**:
   - Enter the encrypted text in the input field.
